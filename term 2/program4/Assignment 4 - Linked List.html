Program Description: Your assignment is to write a linked list class. Your linked list class will store your string class that you wrote for progam 3 . You will need to add three new functions to your string class to make it useable for this assignment ( = =, <, > ).

Your linked list will be a doublely linked list, meaning that it will have links to both the next and previous nodes. Your linked list will also be an ordered list, which means that as you insert new strings into the list, they will need to be inserted in the correct position ( from smallest to largest).

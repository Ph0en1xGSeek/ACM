A.   Sheldon's Friendship I
Time Limit: 1.0 Seconds   Memory Limit: 65536K


Description

In the American movie "Big Bang", Sheldon has 4 friends(Penny, Leonard, Raj and Howard). One day, he want to build a new friendship with one guy named Kripke so that he can use Kripke's new machine (which is called open science grid computer blablablablabla...). But he find it is difficult to maintain five friendships, so, he is going to let one of them go.(What a boy...)

Every friend has a value in Sheldon's mind, and he will delete the one with lowest value. Please help him to make the ridiculous decision.

Input

The fist line is an integer T (T ≤ 20), which is the number of test cases, T cases follows. Each case will contain 4 lines, for each line there are one English name (Penny, Leonard, Raj or Howard) and one value (0 ≤ value ≤ 100). It is guarantee that no two value is the same.

Output

For each case, output the "Poor xxx", the one who has the lowest value, xxx is his/her name.

Sample Input

2
Penny     12
Leonard   32
Raj       43
Howard    33

Penny     12
Leonard   32
Raj       43
Howard    11

Sample Output

Poor Penny
Poor Howard
